[Project Euler](https://projecteuler.net/) problems. For each question 
there is (or will be!) a solution written in:
* Python
* NumPy
* Java 
* Rust
* Julia 
* C
* Golang
* C++
* Bash

My in depth explanations can be found on my [Notion Page](https://ben-crane.notion.site/Benjamin-Crane-1f937b3d86304511bc2276a2b354920d)!
